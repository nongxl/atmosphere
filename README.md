# 🌀 Micro Atmosphere - 3D 气象物理仿真微缩盆景

`Micro Atmosphere` 是运行在 **M5StickS3 (ESP32-S3)** 硬件平台上的气象流体物理仿真系统。
项目利用三维元胞自动机（3D Cellular Automata）与物理动力学守恒定律，在 16x16x12 的三维空间体素网格内，高帧率地模拟了大气对流、饱和冷凝、云雾汇聚、降雨倾泻、三维分形闪电、以及台风气旋风眼的形成与消散过程。

---

## 🚀 核心技术与系统实现

### 1. 🧵 FreeRTOS 异步双核线程解耦 (消除 I2C 阻塞)
* **设计初衷**：外接的 ENV III 传感器（包含 SHT30 与 QMP6988）在进行 I2C 数据读取时存在多达几十毫秒的阻塞延迟（I2C 等待和物理传感器采样响应），导致前台画面每隔 20 秒发生严重掉帧和 IMU 转动顿挫。
* **物理实现**：
  * 在 `sensors.cpp` 中，创建运行在 **Core 0** 上的后台异步任务 `sensorsTask`，每隔 20 秒独立轮询读取外接传感器的温度、湿度和气压。
  * 前台 UI 渲染和 IMU 滤波器运行在 **Core 1**。前台通过非阻塞的互斥共享接口直接提取后台更新的数据，**彻底消除了传感器读取导致的画面卡顿，保证了 60 FPS 极速图形响应**。
  * 主循环尾部强制增加 `delay(2)` 空闲礼让，防止 Core 1 满载导致 ESP32 内部 WiFi 自愈状态机或后台 I2C 线程饥饿。

### 2. ⚡ 3D 自发光分形闪电与多类型渲染
* **物理震源定位**：系统在发生打雷时，先遍历高度 $z \geq 3$ 且密度大于 `0.5` 的云体素，从中随机挑选一个活跃格点作为 **3D 物理震源** $(ex, ey, ez)$，并通过 3D 摄像机投影公式映射得到屏幕 2D 起始坐标，保证电弧百分之百在最厚的云层深处被引燃。
* **体积光三维发光扩散（Mie 漫反射）**：
  * 抛弃了粗暴的全屏填色涂白。云粒子在绘制时计算到 3D 震源的欧氏距离平方 $D^2$，并通过指数衰减函数 $expf(-0.022 \times D^2)$ 融合闪电颜色。
  * 距离震源极近的云层过载暴晒为耀眼纯白色，中等距离平滑插值淡紫/金黄色，远距离不受影响，极具 3D 立体体积发光漫步散射感。
* **三类闪电分流形态**：
  * **云内闪电**：无电弧线条，单纯云层局部爆亮，模拟深层闷雷。
  * **云际闪电**：电弧横向折行击穿云层高度，受高度约束不落地。
  * **闪地闪电**：电弧纵向贯穿直至地面（屏幕下边缘）。
* **多回击频闪 (Stroke Flicker)**：扩展至 5 帧，内置双波峰正弦衰减振幅波形 `1.0 -> 0.25 -> 0.85 -> 0.40 -> 0.15`，还原自然电击的高频跳动。
* **粗细线宽模拟**：主干（`depth >= 2`）采用多像素偏移法重复绘制，模拟粗壮高亮电弧；侧枝（`depth < 2`）采用单像素细线段，构成强烈的粗细对比。

### 3. 🌀 台风（热带气旋）动力学与晴空风眼结构
* **外部能量注入 (麦克风交互)**：向设备麦克风吹气（`M: >65dB`）被定义为外部热带气旋初始扰动能量的注入。台风成长值 `_typhoonGrowth` 启动快速积分爬升；吹气停止后系统由于阻尼耗散自然退化（`0.0 ~ 1.0` 范围）。
* **风眼墙气旋自转 (Eye Wall)**：
  * 当台风成长时，以中心 $(7.5, 7.5)$ 为圆心的 2.0 ~ 5.5 格点环状区域被设定为风眼墙。
  * 注入强烈的对流垂直风速 `velocityZ += 3.6 * factor * _typhoonGrowth`。
  * 强行在旋转切线向量 $\vec{v}_{rot} = (-dy/D, dx/D)$ 上做高达 70% 权重的速度场插值锁定，迫使周围流体和雨滴粒子构成极其急剧的逆时针旋转漩涡。
  * 加速此区域的水汽凝聚效率，合围形成密布的螺旋状黑云墙。
* **晴空风眼沉降区 (Typhoon Eye)**：
  * 在中心 1.8 格以内的风眼洞：
  * 强制注入垂直向下的下沉气流 `velocityZ = -1.3 * (1.0 - D/R_eye) * _typhoonGrowth` 阻断对流。
  * 将格点云密度 `cloudDensity` 乘以平方比率 $(D / R_{eye})^2$ 强行蒸发驱散。使得体积云在渲染时，正中心会**露出一块非常通透、完美的晴空圆洞**。
* **气压梯度力骤降 (Depression Core)**：
  * 台风中心是强低压。我们根据格点到风眼距离，重塑了钟形倒气泡压强分布：
    $$P(D) = P_{base}(z) - 45.0 \times \_typhoonGrowth \times \exp(-0.065 \times D^2)$$
  * 当台风最成熟时，中心压力**跌落达 45 hPa**，并立刻融合反馈在前台显示屏的状态栏 `P` 值上（从常压骤降至 `968 hPa` 等极低压），呈现暴风雨肆虐的真实感。
* **风暴雨丝斜飞与云絮狂舞**：
  * **雨滴斜卷**：下落的雨滴粒子会自动继承并受到网格极速气旋风速的影响，产生横飞和斜向倾洒的物理动态。
  * **云流动加速**：体积云的 Mie 漫反射流动速度在台风发生时自动加速至原本的 **3.5 倍**，表现狂风大作的飞舞撕裂。

### 4. 🧭 IMU 视口全息对齐与自适应 3D 空间采样
* **全息透视窗口**：
  * 纠正了重力轴向绑定：左右偏转（Roll）控制水平旋转 `azimuth`；前后偏转（Pitch）控制仰角 `elevation`。
  * 在两个通道上施加负号以构建**反向视角运动模型**（全息镜面视窗：手势左倾，窗口右转，露出云的左侧侧面），实现完美符合空间透视的镜面深度。
* **自适应 3D 空间跨步采样 (Adaptive Stride)**：
  * 解决 55 帧粒子预算限制下，浓云和台风会独占高层预算，导致中低层和侧边云被全部丢弃，云朵在视觉上失去 IMU 响应的重大 Bug。
  * 当活跃云量极高时，自动将采样步长 `baseStride` 提至 2 甚至 3，并且直接跳过 Z 轴的奇数高度层，将预算点均匀平摊在整个三维体积空间中。**使大云团与台风在偏转设备时呈现出无与伦比的 3D 立体偏转透视响应**。

## 📂 项目结构与核心模块

* **`main.cpp`**：
  * 姿态仪滤波平滑（`ALPHA = 0.18f`），IMU 透视坐标反向映射，传感器任务分发。
* **`sensors.cpp`**：
  * 传感器初始化与基于 Core 0 的 FreeRTOS 多线程异步轮询任务（`sensorsTask`）。
* **`AtmosphereSimulation.cpp`**：
  * 体积云、雨量、气旋漩涡风场、沉降风眼晴空蒸发、及中心钟形低气压物理演变方程。
* **`Renderer.cpp`**：
  * 3D 等轴相机投影、体积云 Mie 散射自发光阻尼漫反射、分形闪电绘制、雨粒子惯性飘飞渲染。
